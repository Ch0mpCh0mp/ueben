Aufgaben zum Lösen

VS Code öffnen und Projekt anlegen

  1. Öffne VS Code über das Terminal
  2. Erstelle ein neues Verzeichnis für dein Projekt
  3. Erstelle eine HTML Datei, eine CSS Datei und eine README Datei im neuen Projektverzeichnis

Inhalte hinzufügen und in Github Repository pushen

  1. Erstelle eine einfache CSS Datei
  2. Schreibe eine kurze Beschreibung in die README.md Datei (z.B. "Mein erstes Projekt")
  3. Verbinde dein lokales Projekt mit einem Github Repository und pushe die Dateien auf Github.

Git Befehle Üben

  1. Stelle sicher das du git init, git status, git add ., git commit -m "Erste Version" und git push origin main beherrschst
  2. Übe das Erstellen eines neuen Branches: git checkout -b neuer-branch
  3. Merge den Branch zurück in main

HTML und CSS Dateien anpassen

  1. Füge zusätzliche HTML Elemente hinzu, wie z.B. ein Bild, eine Liste oder ein Link
  2. Verändere das Layout der Seite mit CSS, z.B. durch Setzen von Padding, Margin oder Textfarbe
